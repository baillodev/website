# BAILLO.DEV

**Portfolio & Personal Website of Mamadou Baillo Diallo**  

Live site: [https://baillo.dev](https://baillo.dev)

---

## Overview

Modern, responsive, production-ready portfolio showcasing:

- Web projects
- Mobile applications
- AI integrations
- Blog for sharing insights and technical articles

Designed with clarity, performance, and maintainability in mind.

---

## Tech Stack

- Next.js (SSR, SSG, routing, API routes)  
- React (hooks, context, performance optimization)  
- TypeScript (type-safe codebase)  
- Tailwind CSS (utility-first, responsive design)  
- shadcn/ui (component-driven UI)

---

## Features

- Fully responsive portfolio  
- Clean, component-driven UI  
- Blog with articles and insights  
- Optimized for performance and maintainability  

---

## Local Setup

```bash
# Clone the repo
git clone https://github.com/baillodev/website.git
cd website

# Install dependencies
npm install

# Run development server
npm run dev

# Open the site locally
# http://localhost:3000
```

---

## Deployment

- Deployed on Vercel  
- Optional: CI/CD can be set up via GitHub Actions for automated builds and deployments
